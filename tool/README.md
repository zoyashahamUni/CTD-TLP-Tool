
NuXmv model and Combinatorial Test Design (CTD) artifacts for my master's thesis.

## Structure
- `model/` — NuXmv models and property files (e.g., `shoppingCartFixed.smv`)
- `ctd/` — CTD test matrices (full & pairwise) and generation scripts
- `scenarios/` — Manual/automated scenario descriptions
- `results/` — Model checker outputs

## Quick Start
1. Open `model/shoppingCartFixed.smv`
2. See `ctd/ctd_matrix_pairwise.csv` for minimal test set
3. Put checker outputs under `results/`
